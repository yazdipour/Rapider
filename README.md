# Rapider

A Robot that plays rapid70, using:
* Python
* OpenCV
* ImageGrab
* numpy
* Tkinter

Get Rapid70 from GooglePlay: https://play.google.com/store/apps/details?id=ab.rapid70&hl=en_US
